SIMON GAME

The Simon game is a classic electronic memory game that tests and improves your memory and concentration skills. The game consists of a circular console with four large colored buttons (red, blue, green, and yellow) and plays sequences of lights and sounds.

DICE GAME

The Dice game is played between only two players ,its like a toss between two players it totally shows random results and is completely unbiased. You could use this game to play other games.

THE DRUM KIT

Virtual drum kit using keyboard keys is a fun way to play drums on a computer or laptop without needing an actual physical drum set. Remember that while playing a virtual drum kit can be enjoyable and educational, it may not provide the same tactile experience as playing an actual drum set. However, it can be a fantastic option for drummers who want to practice quietly or don't have access to a physical drum kit.
